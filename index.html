Google Drive

Connecte ta collection pour activer la synchronisation automatique et le chargement des photos.

Drive connecté
1
Va sur console.cloud.google.com → crée un projet → active l'API Google Drive.
2
Va dans APIs & Services → Identifiants → crée un OAuth 2.0 Client ID de type Application Web. Dans Origines JavaScript autorisées, ajoute l'URL de ton GitHub Pages.
3
Dans Écran de consentement OAuth → publie l'app (sinon limité aux comptes test).
4
Colle ton Client ID ci-dessous et appuie sur Se connecter.

Ton Client ID est stocké localement sur cet appareil uniquement.
Les photos et données sont sauvegardées dans un dossier TCG Vault sur ton Drive.
